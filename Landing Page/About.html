About

The RAISE Playground is a block-based programming platform that we developed to support hands-on learning about AI and robotics for students and beginning programmers. We developed these blocks on a forked version of the open source Scratch Blocks repository.

Anyone is welcome to fork our Github repository per our permitted usage license.
